# Health Insurance Charges Analysis (R Project)

## Objective
Analyze how age, BMI, smoking status and other factors affect insurance charges.

## Tools
- R
- dplyr
- ggplot2

## Dataset
Public health insurance dataset from Kaggle.

## Key Observations
- Smokers have significantly higher insurance charges.
- Insurance costs increase with age and BMI.
- A linear regression model highlights major cost drivers.

## Files
- insurance.R → R code
- insurance.csv → Dataset
- smoker_vs_nonsmoker_charges.png → Output plot
